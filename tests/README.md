# Tests

This directory contains both test data and test cases

## Mobly Tests

1. Install Mobly:

```
pip install mobly
```

2. Install `sample_04_cube` from BigWheels to the device.

3. Turn the device screen on.

4. Run the Mobly script:

```
python gfxr_capture_test.py -c mobly_config.yml
```

On Linux, results are in `/tmp/logs/mobly`
